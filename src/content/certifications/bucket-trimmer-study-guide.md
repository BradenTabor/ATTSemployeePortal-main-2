# Bucket Trimmer Certification — Study Guide

## Overview

This guide covers key topics for the Bucket Trimmer certification test. Use it alongside hands-on training.

## Hardware Identification

Learn to identify these components:

- **Boom** — Main lifting arm
- **Bucket** — Worker platform
- **Upper / lower controls** — Operator stations
- **Outriggers** — Stability supports
- **Hydraulic system** — Power and fluid
- **Insulated section** — Dielectric protection
- **Load chart** — Capacity limits
- **Emergency descent** — Safe lowering
- **Safety harness points** — Fall protection

## Knots and Rigging

Practice these knots:

- Bowline, Clove hitch, Half hitch
- Running bowline, Timber hitch
- Double sheet bend, Figure eight
- Tag line attachment and sling inspection

## Trimmer Observation

- Proper PPE (full-body harness, hard hat, etc.)
- Safe work zone and line clearance
- Communication and emergency procedures
- Pre-use inspection and load handling
- Environmental awareness and traffic control
- Equipment shutdown and documentation

## Practice

Take the certification test from **Resources → Certifications** when you’re ready.
