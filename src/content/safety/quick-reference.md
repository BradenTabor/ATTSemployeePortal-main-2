# Safety Quick Reference

## Before Starting Work

1. **Complete your JSA** — Document hazards and mitigations
2. **Check weather conditions** — Be aware of storms, wind, heat
3. **Inspect equipment** — Verify all gear is in working order
4. **Communicate with crew** — Brief everyone on the work plan

## Emergency Contacts

- **911** — Life-threatening emergencies
- **Company Safety Line** — Report incidents and near-misses
- **Nearest Hospital** — Know the location before starting work
- **General Foreman** — First point of contact for site issues

## Common Hazards

| Hazard | Mitigation |
|--------|------------|
| Electrical contact | Maintain clearance, use insulated tools |
| Falls from height | Use harness, secure anchor points |
| Struck-by objects | Wear hard hat, establish drop zones |
| Traffic | Set up cones, use flaggers |
| Heat stress | Hydrate, take breaks, monitor symptoms |

## Incident Reporting

Report all incidents — including near-misses — within 24 hours:

1. Ensure scene is safe
2. Provide first aid if needed
3. Notify supervisor immediately
4. Complete incident report form
5. Preserve evidence for investigation

## Remember

**When in doubt, stop work and ask.** No task is worth risking your safety.
